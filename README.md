# E-Commerce Sales Analysis 2019

## Project Overview
An end-to-end data analysis project analyzing e-commerce 
electronics sales data for 2019. This project covers data 
cleaning, exploratory data analysis, and interactive 
dashboard building to uncover key business insights.

## Tools Used
- Python (Google Cola)
- Power BI Desktop
- Pandas, Matplotlib, Seaborn

## Dataset
- 12 months of e-commerce sales data
- Nearly 187,000 rows of transaction data
- Source: Kaggle

## Key Insights
- Total Revenue: $34.4 Million
- Best Month: December
- Best City: San Francisco
- Highest Revenue Product: MacBook Pro Laptop
- Most Sold Product: AAA Batteries
- Peak Order Hours: 10am - 1pm

## Project Structure
- data/raw - Original monthly CSV files
- data/processed - Cleaned data
- notebooks - Python analysis notebook
- powerbi - Power BI dashboard
- screenshots - Dashboard preview

## Dataset Source
